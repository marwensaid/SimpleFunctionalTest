SimpleFunctionalTest is sized for maven usage.
If you use another build engine (ant...), you probably want to target other source and class folder.

Change source folder

The default source folder scanned is src/test/java/
Simple functional test will load classes from this folder( for example the class sft.integration.set.SourceAndClassSettings will be loaded from src/test/java/sft/integration/set/SourceAndClassSettings.java)
By setting the source folder property to src/test/other/folder/
Simple functional test will load classes from this folder( for example the class sft.integration.set.SourceAndClassSettings will be loaded from src/test/other/folder/sft/integration/set/SourceAndClassSettings.java)

Change class folder

Class folder allow to define the original project path
The default class folder is target/test-classes/
Project folder is targeted with ../../
By setting the class folder property to bin/
Project folder is targeted with ../
