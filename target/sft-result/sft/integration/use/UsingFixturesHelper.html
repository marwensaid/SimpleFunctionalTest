Re-Use your fixtures between your classes: fixtures helpers

Fixtures could be group into FixturesHelper, and then could be shared between use cases.

Using fixtures helper

To share fixtures between your classes without using inheritance you have to write its in a fixtures helper class
In the use case class add non-public field instantiating this fixtures helper class annotated by @FixturesHelper.
Then all fixtures of this fixtures helper class are seen as fixtures of the use case class and can be used as its own
