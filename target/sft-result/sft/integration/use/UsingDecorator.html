The annotation @Decorate allow to prettify your html report.

Add style

By adding a style decorator with parameter style on a element: @Decorate(decorator = Style.class, parameters ="style")
The element targeted of the html report will have the css class style
Several style can be used together @Decorate(decorator = Style.class, parameters ={"style1","style2","style3"})

The style decorator can be apply on use case, scenario, fixture and sub use case

Group fixtures

By adding a group decorator with parameter When on fixtures: @Decorate(decorator = Group.class, parameters ="When")
Fixtures with the same group name are shown in a specific paragraph with this name as title
java html

Group related use cases

By adding a group decorator with parameter Error cases on field implementing related use case: @Decorate(decorator = Group.class, parameters ="Error cases")
Related use case with the same group name are shown in a specific paragraph with this name as title
java html

Display fixture as table

By adding a table decorator with parameter Addition sample on a fixture: @Decorate(decorator = Table.class, parameters ="Addition sample")
Fixtures call are aggregated in a table
The table title is Addition sample
Column header are field names
Each values are displayed
The issue of each fixture calls is added at each row
